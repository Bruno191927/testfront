# Instalacion
- Ejecutar el comando ```npm install```

# Ejecucion
- Iniciar la aplicacion con ```npm run dev```

# Observaciones
- En caso de que la aplicacion no este respondiendo correctamente(latencia) se debe a que estoy usando un servicio gratuito (railway.com para el backend), se puede usar una base de datos local corriendo el backend de esta aplicaci√≥n. y cambiando su variable de entorno a una coneccion local de mongodb y modificando en este front en la ruta ```src/shared/services/api/category``` y ```src/shared/services/api/service``` por el localhost:3000 ejecutando el backend
